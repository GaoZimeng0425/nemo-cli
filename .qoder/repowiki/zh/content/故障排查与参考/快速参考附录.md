## 命令速查表

本节汇总了 `nemo-cli` 提供的主要命令及其常用子命令和参数，便于快速查阅。

### Git 操作命令 (ng)

`ng` 是用于简化 Git 操作的命令行工具。

| 主命令 | 子命令 | 参数 | 简要说明 |
|--------|--------|------|----------|
| `ng` | `commit` | 无 | 执行 Git 提交操作 |
| `ng` | `pull` 或 `pl` | 无 | 执行 `git pull` 操作 |
| `ng` | `push` 或 `ps` | 无 | 执行 `git push` 操作 |
| `ng` | `checkout` 或 `co` | 无 | 执行分支切换，自动执行 stash 和 pop |
| `ng` | `checkout` | `-b` | 创建并切换到新分支，支持数字自动补全为 `feature/PRIME-{number}` |
| `ng` | `branch` | `delete` | 删除本地分支 |
| `ng` | `branch` | `delete -r` | 删除远程分支 |
| `ng` | `branch` | `clean` | 清理本地已合并的分支 |
| `ng` | `diff` | 无 | 显示工作区与暂存区的差异 |
| `ng` | `merge` | 无 | 执行分支合并 |
| `ng` | `list` | 无 | 列出所有分支 |
| `ng` | `stash` | 无 | 执行 stash 操作 |

## 配置项列表

本节整理了可通过 `configstore` 或环境变量配置的所有选项，包括其作用域、默认值和优先级。

| 配置项 | 作用域 | 默认值 | 优先级 | 说明 |
|--------|--------|--------|--------|------|
| `CONFIG_NAME` | 全局 | `.nemo-cli` | 高 | 配置文件根目录名称，位于用户主目录下 |
| `configRootPath` | 全局 | `~/.nemo-cli` | 高 | 实际配置存储路径，由 `homedir()` 和 `CONFIG_NAME` 组成 |
| `isDebug` | 全局 | `false` | 中 | 通过 `-d` 或 `--debug` 参数启用调试模式 |

## 环境变量说明

本节基于 `.env.example` 文件，详细解释每个环境变量的功能及其安全使用建议。

| 环境变量 | 功能说明 | 安全建议 |
|----------|----------|----------|
| `SLACK_BOT_TOKEN` | Slack 机器人的认证令牌，用于发送消息 | 严禁提交到版本控制系统，应通过 CI/CD 秘密管理 |
| `slack_team_id` | Slack 团队 ID，用于标识目标团队 | 同上 |
| `slack_channel_ids` | Slack 频道 ID 列表，用于指定消息发送目标 | 同上 |
| `API_KEY` | AI 模型 API 密钥（如 OpenAI） | 使用强密钥并定期轮换，避免硬编码 |
| `GOOGLE_API_KEY` | Google 服务 API 密钥 | 限制 API 密钥的使用范围和 IP 白名单 |
| `CONFLUENCE_API_TOKEN` | Confluence API 访问令牌 | 使用专用服务账户，权限最小化 |
| `CONFLUENCE_EMAIL` | Confluence 登录邮箱 | 避免使用个人账户，推荐使用服务账户 |
| `GOOGLE_AUTH_PASS` | Google 应用专用密码 | 启用两步验证后生成专用密码，避免使用主密码 |
| `GOOGLE_AUTH_USER` | Google 账户用户名 | 同上 |
